## GIT ^分布式版本控制系统^

- **与集中式版本控制系统的区别**：  

  ==集中式==：顾名思义，将版本库**集中存放在中央服务器**。工作时从中央服务器获得最新版本，工作后又要把文件推送给中央服务器。最大的缺点就是==需要联网==。

  ==分布式==：工作时无需联网，每个电脑都是完整的版本库。多人协作则采用互相推送的方式。因此若自己电脑损坏不用担心数据丢失，只需要从其他人那里复制，较为安全。

- **特点**：无需联网、安全性高、分支管理强大等
